- les différences avec l'architecture monolithique ❌ / ✔️

Microservice: process isolé qui contient sa propre logique métier et base de donnée etqui communique avec les autres microservices via des API auto générés. 


- la communication asynchrone entre services ❌ / ✔️

Dapr (distributed application runtime): event system pour communication asynchrone découplée. (communique à redis ce qui s'est passé)
Redis: list les evenements qui se sont passé dans les app publisher et annonce aux apps souscrivantes ce qui s'est passé dans les autres apps
